## 👨‍💻 **DEVELOPER STATION**

### **🛸 MISSION PREREQUISITES**

```bash
# Install Rust Toolchain
🦀 https://www.rust-lang.org/tools/install

# Install Node.js Runtime
🟢 https://nodejs.org/en/download/
```

### **🚀 LAUNCH SEQUENCE**

```bash
# 1. Clone the Repository
git clone https://github.com/your-username/orion-calculator.git

# 2. Navigate to Mission Control
cd orion-calculator

# 3. Install Frontend Dependencies
npm install

# 4. Launch Development Server
npm run tauri dev
```

### **🧪 TESTING PROTOCOLS**

```bash
# Navigate to Rust Backend
cd src-tauri

# Execute Test Suite
cargo test

# Expected Output: All systems operational ✅
```

### **🏭 PRODUCTION BUILD**

```bash
# Compile for Production Deployment
npm run tauri build

# Output: Optimized binary ready for distribution 🚀
```
